# Vehicle Suspension Simulator

## Overview
This project models a quarter-car suspension system using Newton's Second Law.

The governing equation:

m x'' + c(x' - r') + k(x - r) = 0

The system is solved numerically using the Runge-Kutta 4 method.

## Features
- RK4 numerical integration
- Suspension animation
- Frequency response analysis
- Natural frequency and damping ratio calculation

## Engineering Concepts
- Second-order differential equations
- Damping ratio classification
- Resonance behavior
- Dynamic systems modeling
